# Intermediate Tier Projects

This folder contains intermediate-level backend projects.  
Focus on **authentication, Docker, caching, real-time systems, and deployment readiness**.

## Projects Included

| Project Name | Description | Key Technologies |
|-------------|-------------|----------------|
| GitHub Trending CLI | Fetch trending repositories | Node.js, Axios, CLI |
| TMDB CLI Tool | Movie info from TMDB API | Node.js, CLI, API Integration |
| Caching Proxy | Proxy server with Redis caching | Node.js, Express, Redis |
| Markdown Note-taking App | REST API for notes using Markdown | Node.js, Express, MongoDB |
| URL Shortening Service | Shorten URLs via API | Node.js, Express, MongoDB |
| Broadcast Server | WebSocket-based message broadcast | Node.js, WebSockets |

## Notes

- Implements **authentication, caching, and error handling**.  
- Projects include **Docker support** for local development.  
- Designed for **production-readiness concepts** like environment variables and logging.
