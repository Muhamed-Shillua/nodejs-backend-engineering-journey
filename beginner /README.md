# Beginner Tier Projects

This folder contains foundational backend projects using Node.js.  
Designed for learning **CLI tools, basic REST APIs, and core backend concepts**.

## Projects Included

| Project Name | Description | Key Technologies |
|-------------|-------------|----------------|
| Task Tracker CLI | Track and manage tasks via terminal | Node.js, CLI |
| GitHub User Activity | Fetch GitHub user data via API | Node.js, Axios, CLI |
| Expense Tracker | Simple CLI for personal finances | Node.js, CLI, File System |
| Number Guessing Game | Interactive number game in terminal | Node.js, CLI |
| Unit Converter | Convert units via web or CLI | Node.js, Express |
| Personal Blog | Simple blog backend | Node.js, Express, File System |

## Notes

- All projects follow basic **Node.js principles**.  
- Focus on **understanding asynchronous code, modules, and CLI interaction**.  
- Each project includes a dedicated README with setup instructions.
